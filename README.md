# Gulp starter for front end static site builds.

Includes:

- Nunjucks for templating
- Autoprefixer
- Sourcemaps
- Babel (for ES6)
- SVG Sprites
- Minifcation and Concatenation for JS and CSS
- Basic HTML layout page
- Bootstrap 4 Alpha 6 scss
- jQuery 3.2.0

Download files and run 'npm install'
